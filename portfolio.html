Vibe Tunes

É uma plataforma para criação, gerenciamento e compartilhamento de playlists musicais. Os usuários podem criar suas próprias playlists, adicionar músicas a elas e compartilhá-las com outros usuários. Usuários comuns podem criar contas, alterar seus dados, criar playlist. Usuários administradores podem excluir usuários da plataforma, cadastrar novas músicas, cadastrar artistas. Feito utlizando Java JDK 21 e spring boot. Confira o código fonte aqui.
